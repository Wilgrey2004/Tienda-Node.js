## Tienda Virtual de Node.js

## Descargar las dependencias
```shell
  npm install
```

## Correr la aplicacion en el puerto 3000 o 3001

```shell
  npm run dev
```

## Correr el Tailwindcss en dev 

```shel
  npm run css
```
